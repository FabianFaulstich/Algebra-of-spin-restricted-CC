Algebra of Spin-Restricted Coupled Cluster (SR-CC) Theory

This project uses a Python virtual environment and pyproject.toml for dependency management. Follow the steps below to install the package and its dependencies.

To set up your code locally, open a terminal and follow the instructions:

1. Clone the repository:
   git clone https://github.com/<your-username>/Algebra-of-spin-restricted-CC.git
   cd Algebra-of-spin-restricted-CC

2. Create a virtual environment:
   python3 -m venv srcc

3. Activate the environment:
   source srcc/bin/activate

4. Set up the environment:
   pip install -e .
